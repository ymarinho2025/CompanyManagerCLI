# CompanyManagerCLI
## Sobre o modelo CLI

Este projeto foi desenvolvido utilizando o modelo **CLI (Command Line Interface)**, 
ou seja, uma **Interface de Linha de Comando**.

Isso significa que o sistema é executado diretamente no terminal (console), 
onde o usuário interage digitando comandos e selecionando opções numéricas.

### O que é CLI?

CLI (Command Line Interface) é um tipo de interface baseada em texto, 
onde a comunicação com o sistema ocorre por meio de entradas digitadas no terminal.

Exemplos conhecidos de aplicações CLI:
- Git
- Docker
- AWS CLI
- Python no terminal
- Prompt de Comando (CMD)
- PowerShell

### Por que usar CLI neste projeto?

O uso de CLI permite:

- Foco total na lógica do sistema
- Melhor compreensão de estruturas de dados
- Aprendizado de controle de fluxo
- Desenvolvimento de sistemas leves e rápidos
- Facilidade de execução sem dependência gráfica

### Evolução futura

O sistema pode futuramente evoluir para:

- GUI (Interface Gráfica)
- Aplicação Web
- API REST
- Integração com banco de dados

Esta versão representa a base estrutural do sistema.
